# Вариант задания: 1.a - Прямоугольник

# Описание программы
Программа вычисляет основные параметры прямоугольника:
- Периметр
- Площадь
- Длину диагонали

# Как использовать
1. Запустить программу
2. Ввести значения:
   - length - длина прямоугольника
   - width - ширина прямоугольника
3. Получить результаты вычислений
